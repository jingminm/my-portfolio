Beyster 1695: The Staff Maintenance Simulator

This project transforms a virtual recreation of the Beyster 1695 laboratory into a reactive "Staff Simulator" where players utilize a custom affordance-based interaction system to resolve environmental disturbances, manage NPC requests, and trigger dynamic audio-visual rewards.

Gameplay Interaction components

  • NPC Dialogue & Engagement
  • Interactive Whiteboard Drawing
  • Copy Machine Physical Duplication
  • Fire Alarm & Light Toggle
  • Environment Day-Night Progression
  • Trash Disposal System
  • Immersive Audio-Visual Feedback

Technical Implementation

  • Custom Affordance Actor Component system
  • Global TaskCounter variable managed within the VRGameMode
  • LineTraceByChannel and Break Hit Result for precise interaction
  • Event Tick combined with Modulo and Random Float nodes for automated light flickering and SkySphere Sun Height "chasing" logic
  • SpawnActorFromClass for duplicating objects and rewards
  • SetWorldRotation paired with FindLookAtRotation for NPC head-tracking
  • Toggle Visibility for state-based actors
  • Widget Components updated via Casting to display dynamic 3D dialogue
  • RefreshMaterial() function called to synchronize atmospheric skybox with shifting day-night cycle
